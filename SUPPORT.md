# Support

## Questions and usage help

- Open a discussion or issue in the repository:
  - https://github.com/endalk200/better-webhook/issues

## Bug reports

When filing an issue, include:

- Package/CLI version
- Node/Bun/runtime version
- OS and architecture
- Reproduction steps
- Expected vs actual behavior

## Security issues

For vulnerabilities, do not open a public issue.
Follow the process in `SECURITY.md`.
