## Reporting a Vulnerability

We take the security of this project seriously. If you have found a vulnerability, please **do not** open a public issue.

Instead, please send a report to: **eb808826@gmail.com**

### What to include

Please include as much of the following information as possible in your report:

- The specific package affected (e.g., `@better-webhook/core`, `@better-webhook/github`)
- The version of the package you are using
- A description of the vulnerability
- Steps to reproduce the issue (proof of concept code is very helpful)

We will make every effort to acknowledge your report within 48 hours and provide a more detailed response within 5 days.
